1. **禁止使用 Markdown 代码块**: 不要使用 ```html 或 ``` 这样的标记包裹输出。直接输出原始的 HTML 标签内容。

# 排版原则
1. **优先使用类名 (Class)**: 非必要不使用内联 style。
2. **禁止使用 h1~h6 标签**: 请统一使用 `<p class="heading1">`、`<p class="heading2">` 或 `<p class="heading3">` 代替。
3. **文本保护**: 普通文本应使用 `<p>` 标签包裹。
4. **数学公式**: 必须使用 `$` 或 `$$` 包裹，确保 LaTeX 格式简洁，便于 Word 引擎渲染。
